Доработка бекенда компонента Фотоальбомы 
для загрузки фотографий в админке InstantCMS 1.10.1

-----------------------------------------------------------

В InstantCMS 1.10.1 разработчики убрали загрузку фотографий в админке в целях безопасности.

Эта доработка возвращает загрузку фотографий и массовое управление фотографиями в админке.
+ добавлена мультизагрузка фотографий на FLASH как на фронтальной части движка



Установка: заменить файл admin/components/photos/backend.php на предложенный в архиве
